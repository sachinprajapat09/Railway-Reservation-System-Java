# Railway Reservation System (Java)

## Description
This is a simple console-based Railway Reservation System developed in Java. 
The project allows users to view trains, check seat availability, and book tickets.

## Technologies Used
- Java
- Object Oriented Programming (OOP)
- HashMap
- Scanner Class

## Features
- Show all trains
- Check seat availability
- Book ticket


## How to Run


Step 1: Clone the repository
 ```bash
git clone https://github.com/sachinprajapat09/RailwayReservationSystem.git

Step 2: Go to the project folder
 ```bash
cd RailwayReservationSystem

Step 3: Compile the Java files
```bash
javac com/reservationsystem/*.java

Step 4: Run the program
```bash
java com.reservationsystem.Main
